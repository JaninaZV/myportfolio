What I’m Building Now

WATERMARK

OPEN SOURCE · METHODOLOGY-FIRST · v0.1

● ACTIVE DEVELOPMENT

Watermark is my current focus — an open-source CLI that measures the real energy, carbon, and water footprint of AI workloads. Not a guess from cpu_percent, but hardware reads where possible (RAPL, NVML), regional grid data where we must model, and every number tagged so reviewers can audit or challenge any assumption. Each run ships measurements.csv, summary.json, and interactive HTML dashboards — plus a portfolio view to compare the same workload across regions.

⚡ ENERGY RAPL · NVML · GPU
🌫 CO₂e GRID CARBON
💧 H₂O DIRECT + INDIRECT
◎ STRESS WATERSHED
⌗ AUDIT SOURCE TAGS
